# Enhanced Survival Options
サバイバルモードを快適にプレイするための設定項目を追加するデータパックです。

## 【データパック名】
Enhanced Survival Options Ver.1.0.0

### 《対応バージョン》
 - Minecraft Java Edition Ver.1.21.11
 - 他のデータパック、MODと併用する際の挙動は保証しません。

## 【主な機能】
 - 天候の操作

順次機能を追加する予定です。

## 【操作方法】
 - 適切に導入が完了している場合、ワールドに入ったタイミングなどリロードがかかるタイミングでメッセージが表示されます。<br>
そこに記載されているキー（割り当てを変更していない場合はGキーです）を押すことで、当データパック専用のメニューを開くことができます。
